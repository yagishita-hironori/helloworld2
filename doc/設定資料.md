## 1. ローカル開発環境

### 前提条件

| ツール | バージョン |
|---|---|
| .NET SDK | 8.0.419 |
| Node.js | 20 以上 |
| Docker Desktop | 任意 |
| AWS CLI | 任意 |

### 注意事項

- Windows 環境では x64 版 (`C:\Program Files\dotnet`) が PATH で x86 版 (`C:\Program Files (x86)\dotnet`) より先に来るよう設定すること
- NuGet ソースが未設定の場合は以下を実行する

```bash
dotnet nuget add source https://api.nuget.org/v3/index.json --name nuget.org
```

### 起動手順

**バックエンド (ターミナル 1):**

```bash
cd backend
dotnet run
# http://localhost:5000 で起動
```

**フロントエンド (ターミナル 2):**

```bash
cd frontend
npm install
npm start
# http://localhost:5173 で起動 (Vite)
```

### ローカル API プロキシ設定

`frontend/vite.config.js` にプロキシを設定済み。フロントエンドの `/api/*` リクエストはバックエンド (`http://localhost:5000`) に転送される。

## 4. 手動デプロイ手順

### フロントエンド

```bash
cd C:\repos\beans-web-test

# ECR ログイン
aws ecr get-login-password --region ap-northeast-1 | docker login --username AWS --password-stdin 869935101124.dkr.ecr.ap-northeast-1.amazonaws.com

# ビルド & push
docker build -t helloworld-frontend ./frontend
docker tag helloworld-frontend:latest 869935101124.dkr.ecr.ap-northeast-1.amazonaws.com/helloworld-frontend:latest
docker push 869935101124.dkr.ecr.ap-northeast-1.amazonaws.com/helloworld-frontend:latest

# ECS 再デプロイ
aws ecs update-service --cluster helloworld-cluster --service helloworld-frontend-service --force-new-deployment --region ap-northeast-1
```

### バックエンド

```bash
cd C:\repos\beans-web-test

# ビルド & push
docker build -t helloworld-backend ./backend
docker tag helloworld-backend:latest 869935101124.dkr.ecr.ap-northeast-1.amazonaws.com/helloworld-backend:latest
docker push 869935101124.dkr.ecr.ap-northeast-1.amazonaws.com/helloworld-backend:latest

# ECS 再デプロイ
aws ecs update-service --cluster helloworld-cluster --service helloworld-backend-service --force-new-deployment --region ap-northeast-1
```

### デプロイ確認

```bash
aws ecs describe-services `
  --cluster helloworld-cluster `
  --services helloworld-frontend-service helloworld-backend-service `
  --region ap-northeast-1 `
  --query "services[*].{Name:serviceName,Running:runningCount,Desired:desiredCount}"
```
